Task001 - Magic Operation

Task:
We have an integer array of elements example -> [1, 4, 8]
If we have in the array more than 5 elements, we summarize all elements,
If in array 3-5 elements we subtract all values,
If in array 2 elements we divide elements
And if in array 1 element we get it cubed like 3^3 = 27 or 2^3 = 8
The last thing, if the array is empty we return a string "EMPTY ARRAY"

Naming:
Object name -> Task001
Method name -> doOperation(<array>) -> expect return one value.
