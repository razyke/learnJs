Task011 - Arithmetic numbers to rome numbers

Task:
We must convert arithmetic number to rome number.

Constraints: 1 < n < 100

Example:
convertToRomeNumber(4) -> "IV"
convertToRomeNumber(37) -> "XXXVII"
convertToRomeNumber(83) -> "LXXXIII"

Naming:
Object name -> Task011
Method name -> convertToRomeNumber(<number>) -> expect return <string> rome number
